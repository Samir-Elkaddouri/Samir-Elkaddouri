## À propos

Technical Project Manager Web en alternance, je fais le pont entre les développeurs et les clients pour le déploiement de stratégies d'optimisation digitale (A/B testing). Au quotidien, je développe et débugge des composants front-end (JavaScript, CSS), je pilote la QA jusqu'à la mise en production, et je coordonne des tests de bout en bout.

Mon profil est volontairement double : gestion de projet et exécution technique. Au-delà de mon poste, je construis des projets personnels (outils statistiques, automatisations, extensions navigateur) pour approfondir mon identité de développeur. Je me forme en continu vers un parcours plus technique.

```yaml
Rôle:        Technical Project Manager Web (alternance)
Spécialité:  A/B Testing & CRO · Front-End
Approche:    du besoin métier à la mise en production
Open to:     alternance · projets front-end · collaborations techniques
```
